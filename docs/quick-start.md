# 🚀 Quick Start Guide

Get up and running with HotPlex in 5 minutes.

---

## 🛠️ Step 1: Choose Your Installation Path

HotPlex supports two primary ways to run:

### Option A: Docker Container (Recommended ⭐⭐⭐)

**Best for**: Production deployment, isolated development, multi-bot orchestration (1+n Matrix).
- **Advantages**: Out-of-the-box, includes full 2026 toolchain (`uv`, `bun`, `trivy`), guaranteed environmental consistency.
- **Get Started**: Jump directly to the [**Docker Deployment Guide**](../docker/README.md).

---

### Option B: Local Binary / SDK (⭐⭐)

**Best for**: Source code customization, embedded integration, rapid prototyping.
- **Prerequisites**:
  1. **Go 1.25+** installed.
  2. **Claude Code** or **OpenCode** CLI installed and authenticated.

#### 1. Fast Build
```bash
git clone https://github.com/hrygo/hotplex.git
cd hotplex
make build
```

#### 2. Configure
```bash
cp .env.example .env
# Edit .env with your Slack/Feishu credentials
```

#### 3. Start Engine
```bash
./dist/hotplexd --config configs/chatapps
```
