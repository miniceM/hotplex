# Security Policy

## Supported Versions

We recommend always using the latest version of HotPlex.

| Version  | Supported          |
| -------- | ------------------ |
| v0.3.0   | :white_check_mark: |
| < v0.3.0 | :x:                |

## Reporting a Vulnerability

If you've found a security vulnerability in HotPlex, please do not open a public issue. Instead, please report it privately.

You can report vulnerabilities by:
- Opening a private security advisory on GitHub (if available).
- Contacting the maintainers directly via email (to be configured).

We will acknowledge your report within 48 hours and provide a timeline for a fix. Please give us reasonable time to investigate and address the issue before making any information public.
